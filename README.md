# Atividades M02

Este repositório é destinado à organização das atividades ponderadas do **Módulo 2** do **Inteli — Instituto de Tecnologia e Liderança**.

---

## Sobre o módulo

O Módulo 2 tem como projeto central o desenvolvimento de uma **Aplicação Web** — um sistema com back-end e front-end, focado em cadastro e recuperação de dados para aplicações corporativas. Os conteúdos estudados ao longo do módulo se distribuem em cinco eixos:

| Eixo | Tópicos |
|---|---|
| **Matemática / Física** | Integral Indefinida, Integral Definida, Integração Numérica, Lógica Proposicional, Grafos e Árvores, Estatística Descritiva |
| **Computação** | Bancos de Dados, Padrão MVC, HTML / CSS / JavaScript, Node.js, Testes Unitários, Fundamentos de Redes |
| **Design** | Personas, Prototipação, Design de Interfaces, Usabilidade, Acessibilidade, Testes com Usuários |
| **Negócios** | Ciclo de Vida do Produto, Design Thinking, 4 P's do Marketing, Branding, PMI / PMBok / Agile |
| **Liderança** | Liderança em Situações de Conflito, Vieses Cognitivos, Empatia |
